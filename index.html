Currently, working as Graduate Student Researcher at WiSSR Lab, Department of Computer Science, University of Maryland advised by Prof (Dr.) Nirupam Roy. My research is focused on Embedded System Security and Honeypots for IoT/Hardware devices.

I was working as an "Intern-Product and Project" at OpenEyes Technologies, Inc for a period of 4 months (pretty long period !). For our project, we used Flask, Python, AWS for Backend and ResNet50 ("respect +" for residual learning !) model for Transfer Learning to predict the frames.

Moreover, we implemented the whole application in Docker container (era of virtualization is in jeopardy !) for deployment. We made linux (thanks to linus torvalds !) distro, Debian based Ubuntu container as a base image and using Docker-Compose, we made it possible to make the Mongodb database offline. In addition, to make the prediction at lighting speed (metaphor !) we also made it possible to run the application on top of the NVIDA-GPU (i hope the machine has one !), which makes the prediction at lighting speed (again metaphor but it's not an euphemism !).
